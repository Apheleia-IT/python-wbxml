python-wbxml
============

Python wrapper for libwbxml.

.. code-block:: python

  >>> import wbxml
  >>> wbxml.xml_to_wbxml(xml) #converts xml to wbxml
  >>> wbxml.wbxml_to_xml(w) # converts wbxml to xml

Installation
------------

Just type in your shell:

.. code-block:: bash

  $ pip install wbxml
